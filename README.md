# Проект III. 5-минутная пешеходная доступность "зеленых" зон в Ново-Савиновском районе г. Казани

Ссылка на картосхему: https://nnvorontsova.github.io/Folium-map/

## О проекте

**Выполнила:** Воронцова Наталия, студентка 1 курса ЦУиАГ.

**Цель:** проанализировать 5-минутную пешеходную доступность "зеленых" зон в Ново-Савиновском районе г. Казани для местных жителей и пешеходов в целом.

**Задачи:** 
1) Отметить границы района.
2) Отметить водные объекты и крупное озеленение (без учета газона, кустарников и др.) - "зеленые" объекты.
3) Выделить зоны 5-тиминутной пешей доступности до "зеленых" объектов по пешеходным дорогам.
4) Добавить слой с жилыми домами, чтобы проанализировать, насколько доступны "зеленые" объекты местным жителям.
5) Добавить слой с автобусными остановками, чтобы проанализировать, достаточно ли 5 минут человеку, приехавшему на автобусе, чтобы дойти до ближайшего "зеленого" объекта.
6) Построить и оформить картосхему для визуализации.
7) Произвести аналитику на основе картосхемы.

## Что изображено на картосхеме

1) Подложка cartodbpositron.
2) Границы Ново-Савиновского района г. Казани (все объекты ниже - в границах района).
3) Слои с крупным озеленением (без учета газона, кустарников и др.) и водными объектами.
4) Зоны 5-минутной пешеходной доступности объектов, указанных в п. 3.
5) Жилые дома (при наведении курсора можно ознакомиться с адресом).
6) Автобусные остановки (кластеризованы).
